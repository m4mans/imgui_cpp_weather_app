## Features

- Live animated sky — sun rays, drifting clouds, rain, snow, fog, lightning
- 6 cities: London, Tokyo, Dubai, NYC, Mumbai, Reykjavik
- 7 weather conditions with unique animations per condition
- 5-day forecast + hourly temperature bar chart
- Air quality index tracker
- Toggle between °C and °F
- Custom Dracula-inspired dark theme
